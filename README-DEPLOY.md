## Quick Start

1. Clone and install:
   ```bash
   git clone https://github.com/iamaanahmad/securepass-ai.git
   cd securepass-ai
   npm install
   ```

2. Set up environment:
   ```bash
   cp .env.example .env.local
   # Add your Google Gemini API key to .env.local
   ```

3. Run development server:
   ```bash
   npm run dev
   ```
